Removes isolated vertices from the graph, generated from the background for instance, for pre-processing to the isoperimetric graph partitioning algorithm. Optionally, returns a vector mapping vertices from graph to the result graph so the transformation can be undone. Assumes a bidirectional representation for the graph.

Parameters
graphthe graph to remove isolated vertices from.
vertexMapa vector of size n where n is the number of vertices in graph. Will be populated with the new index for each vertex, or -1 if it was removed.

Removes isolated vertices from the graph, keeping track of which were removed so they can be readded later. Typically used as a pre processing step to segmentation, to remove isolated vertices from the background for instance.
