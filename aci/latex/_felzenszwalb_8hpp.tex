Combines segmentations of the same graph by the following rule\-: two neighboring vertices in the graph are in the same component iff they are in the same component in all segmentations. Useful for combining segmentations of separate channels of a color image.

Weights are ignored, only the graph structure is kept -\/ it is assumed the segmentations were done on the same unweighted graph.
