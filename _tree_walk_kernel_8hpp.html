Computes the tree walk kernel between two labelled graphs given a basis kernel function. It is assumed that each graph is undirected, with each edge duplicated in both the source and destination's adjacency list. Both graphs must be planar.

Parameters
basisKernelbasis kernel function, taking 2 labels as parameters, with the associated area for each segment.
